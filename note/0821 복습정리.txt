[javascript 기본문법]

 - 자료형 : String, Number, Boolean, Object(Array), Function, undefined
 - 내장함수 : alert, prompt, confirm

표현식과 문장 : 하나 이상의 표현식이 모여 문장. 한 문장 단위로 세미콜론으로 구분
식별자 : 이름을 붙일 때 사용하는 단어 (예) 변수명, 함수명, 속성, 메소드
	- 예약어 사용불가, 첫글자는 알파벳으로 시작

 - 변수선언
 var : 재선언 가능. 전역변수로 사용
 let : 재선언 불가. 지역변수로 사용
 const : 값변경 불가
