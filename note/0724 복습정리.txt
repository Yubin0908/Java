[단일행함수 - 문자, 숫자, 날짜형 변환함수]
오직 단일행에서만 적용가능하고, 행별로 하나의 결과를 리턴

※ DUAL 테이블(DUMMY TABLE)
 - 구조 : 테스트 용도의 ORACLE 내장 객체. 한 행으로 출력됨

1. 숫자 함수
TRUNC : 지정한 자릿수 이하 숫자 버림

2. 문자 함수
LOWER : 소문자로 표시
UPPER : 대문자로 표시
SUBSTR : 문자를 시작위치 ~ 추출할 문자수를 지정해 추출
LENGTH : 문자길이 추출
LPAD, RPAD : 입력받은 문자열과 기호를 정렬하여 특정 길이의 문자열로 반환
TRIM : 문자를 잘라낸뒤 남음 문자를 표시
REPLACE : 문자열에서 특정문자로 변경

4. 날짜 함수
SYSDATE : 시스템 기준 현재 날짜를 반환
MONTHS_BETWEEN : 두 날짜 사이의 개월수 반환
