  ☆ CSS 선택자

 [1] css 선택자란? 
 - HTML 문서 중 특정 부분을 선택하여, 해당 부분의 속성을 변경하는 목적으로 사용

(1) tag 선택자 : 특정 태그를 선택하여 CSS속성을 적용할 수 있다.
(2) * 선택자 : 전체 선택자(문서 전체를 선택하는 의미)
(3) id, class 선택자 : 태그의 id/class를 이용하여 css속성을 적용할 수 있다.
(4) 혼합 선택자 : tag, id, class를 혼합하여 css속성을 적용할 수 있다.
(5) 속성 선택자 : tag의 속성을 선택하여 css속성을 적용할 수 있다.
(6) 후손/자손 선택자 : tag의 속성을 선택하여 css의 속성을 설정할 수 있다.
(7) 동위 선택자 : ta의 동등위치를 판단하여 css의 속성을 설정할 수 있다.
(8) 반응 선택자 : 마우스 over 시, 반응에 따른 속성을 설정할 수 있다.
